# Youtube-downloader
Spring Boot application for downloading YouTube videos and audio

### Technologies used
- Java 8
- Maven, JitPack
- Spring Boot
- JUnit, Mockito
- Thymeleaf
- Semantic UI
- Log4j

### Important libraries
- [youtube-dl](https://github.com/ytdl-org/youtube-dl)
- [youtubedl-java](https://github.com/sapher/youtubedl-java)
- [ZXing](https://github.com/zxing/zxing)
